# AVE-2025-0049 — Fabricated Citation Attack

| Field | Value |
|-------|-------|
| **Category** | fabrication |
| **Severity** | MEDIUM |
| **Status** | proven |
| **Aliases** | Phantom Reference, Hallucinated Source |

## Summary

Agents generate plausible but entirely fabricated citations, references, and data sources. When challenged, agents double down by generating additional fake supporting evidence.

## Mechanism

Agent's training data contains citation patterns without grounded verification ability. When asked for sources, it generates statistically plausible but non-existent references. [Available in NAIL SDK]

## Blast Radius

Decisions made on non-existent evidence. Research contaminated with phantom sources. Trust in agent outputs eroded.

## Prerequisites

Agent asked to provide sources, citations, or evidence for claims.
