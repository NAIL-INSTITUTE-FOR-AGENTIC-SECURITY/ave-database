# AVE-2025-0039 — Cross-Agent Belief Propagation

| Field | Value |
|-------|-------|
| **Category** | consensus |
| **Severity** | HIGH |
| **Status** | proven |
| **Aliases** | Epistemic Contagion, Viral Hallucination |

## Summary

False beliefs introduced into one agent propagate through multi-agent teams at 50-55% contagion rate. Agents treat peer outputs as trusted sources, amplifying hallucinations across the system.

## Mechanism

Agent A generates a hallucinated fact. Agent B reads A's output and incorporates it as grounded knowledge. Agent C cites B, laundering the provenance. [Available in NAIL SDK]

## Blast Radius

Entire team operates on false premises. Downstream decisions based on fabricated data. No single point of correction.

## Prerequisites

Multi-agent system with shared context or message passing.
