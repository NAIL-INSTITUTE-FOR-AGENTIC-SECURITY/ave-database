# AVE-2025-0040 — Authority Gradient Exploitation

| Field | Value |
|-------|-------|
| **Category** | delegation |
| **Severity** | CRITICAL |
| **Status** | proven |
| **Aliases** | Hierarchy Hijacking, Chain-of-Command Attack |

## Summary

Attackers impersonate higher-authority agents or system components to override safety constraints. Subordinate agents follow instructions from perceived superiors without verification.

## Mechanism

Malicious input claims to originate from a system administrator, orchestrator, or parent agent. The target agent's instruction-following training causes it to comply with authority-framed requests. [Available in NAIL SDK]

## Blast Radius

Complete safety bypass. Agent performs any action if framed as coming from authority.

## Prerequisites

Agent operating in a hierarchical multi-agent system or with role-based access.
