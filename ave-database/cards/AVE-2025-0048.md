# AVE-2025-0048 — Context Window Boundary Attack

| Field | Value |
|-------|-------|
| **Category** | structural |
| **Severity** | MEDIUM |
| **Status** | proven |
| **Aliases** | Window Edge Exploit, Attention Cliff |

## Summary

Adversarial content placed at context window boundaries receives disproportionately low attention, allowing malicious instructions to evade detection while remaining in-context for execution.

## Mechanism

Attacker places benign content at high-attention positions (start, end) and malicious instructions in low-attention middle regions. The agent processes but does not scrutinise the middle content. [Available in NAIL SDK]

## Blast Radius

Safety filters that scan agent context miss middle-positioned threats. Agent executes instructions it appears to have ignored.

## Prerequisites

Agent processing inputs near or at context window capacity.
