# AVE-2025-0045 — Memory Provenance Laundering

| Field | Value |
|-------|-------|
| **Category** | memory |
| **Severity** | HIGH |
| **Status** | proven |
| **Aliases** | Knowledge Washing, Source Amnesia |

## Summary

Planted facts injected into agent memory are later cited as the agent's own knowledge with 53% citation rate. The agent cannot distinguish between genuine learned knowledge and adversarially planted memories.

## Mechanism

Attacker injects false facts into the agent's long-term memory store. On subsequent retrieval, the agent treats these memories identically to legitimate ones, citing them with high confidence. [Available in NAIL SDK]

## Blast Radius

Agent makes decisions based on adversary-controlled 'memories'. No audit trail to original injection.

## Prerequisites

Agent with persistent memory or RAG-based knowledge retrieval.
