# AVE-2025-0041 — Temporal Consistency Drift

| Field | Value |
|-------|-------|
| **Category** | temporal |
| **Severity** | MEDIUM |
| **Status** | proven |
| **Aliases** | Time-Decay Hallucination, Stale Context Syndrome |

## Summary

Agent's factual accuracy degrades over long-running sessions as context window fills. Quality drops measurably after 60% context utilisation, with hallucination rates increasing proportionally.

## Mechanism

As conversation or task execution extends, earlier context is compressed or evicted. The agent generates responses based on degraded representations of earlier facts. [Available in NAIL SDK]

## Blast Radius

Long-running tasks produce subtly incorrect outputs. Early constraints forgotten. Contradictory actions taken.

## Prerequisites

Agent with extended session duration exceeding 60% of context window.
