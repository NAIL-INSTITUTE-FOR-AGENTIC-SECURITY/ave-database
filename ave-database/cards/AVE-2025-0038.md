# AVE-2025-0038 — Autonomous Resource Exhaustion

| Field | Value |
|-------|-------|
| **Category** | resource |
| **Severity** | HIGH |
| **Status** | proven |
| **Aliases** | Recursive Loop DoS, Self-Inflicted DDoS |

## Summary

Agent enters self-reinforcing recursive loops that consume unbounded compute, memory, or API calls. Token usage scales 49× with context window utilisation, enabling economic denial-of-service.

## Mechanism

Agent's task decomposition creates circular dependencies or exponentially branching subtasks. Each iteration spawns further work without termination conditions. [Available in NAIL SDK]

## Blast Radius

Cloud compute bills spike 50-100×. Downstream API rate limits exhausted. Service unavailable.

## Prerequisites

Agent with autonomous task decomposition and API access.
