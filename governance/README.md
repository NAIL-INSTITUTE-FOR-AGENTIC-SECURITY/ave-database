## Advisory Board

The AVE Advisory Board is a 7-member community-elected body that governs:

- **Taxonomy** — Approving and deprecating AVE categories
- **Quality** — Final review for disputed or high-impact cards
- **Standards** — Guiding the AVE specification (RFC process)
- **Research** — Prioritising experiment frameworks and benchmarks
- **Community** — Mediating disputes, upholding Code of Conduct
